Css Selactor Example


Univarsal selctor(*)

*{ font-family: 'Lucida Sans', 'Lucida Sans Regular', 'Lucida Grande', 'Lucida Sans Unicode',
Geneva, Verdana, sans-serif; }

The universal selector is written as * and it means “apply this style to every element.”


Element Selctor

h1{ color: brown;
background-color: whitesmoke;
text-align: center; }

An element selector in CSS is used to select a specific HTML tag and apply style to it.


Class Selctor(.)

.first{ color: aquamarine; background-color: brown; }

Class selector (.) is used to style multiple elements with the same class name


Id Selctor(#)

ID selector (#) is used to style one specific unique element.


Group selector

td,th{ border: 2px solid palevioletred; }

Group selector (,) is used to apply same style to multiple elements together.

Css Descendant Selector

table tr{ background-color: blanchedalmond; }

Descendant selector (space) selects elements inside another element.
